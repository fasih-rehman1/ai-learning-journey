# 1. AI Engineer

**Job Description:**
Design, develop, and deploy AI/ML models to solve real-world business problems.

**Expected Skills:**

* Python
* Machine Learning
* Deep Learning
* TensorFlow/PyTorch
* Data Analysis
* Model Deployment

**Projects:**

* Image Classification System
* Recommendation System
* Predictive Analytics Model

# 2. RAG (Retrieval-Augmented Generation) Engineer

**Job Description:**
Build AI applications that retrieve information from knowledge bases and generate accurate responses using LLMs.

**Expected Skills:**

* Python
* LangChain/LlamaIndex
* Vector Databases (Pinecone, Chroma, FAISS)
* Embeddings
* Prompt Engineering
* API Integration

**Projects:**

* Document Q&A Chatbot
* PDF Knowledge Assistant
* Enterprise Search System
