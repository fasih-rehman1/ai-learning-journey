# Attention Intuition Exercise

Sentence:

Ahmed bought a laptop.

He repaired it because it was slow.

Connections:

He → Ahmed

First it → Laptop

Second it → Laptop

Observation:

Attention helps the model connect pronouns to the correct nouns.

Without attention, the model may misunderstand references and generate incorrect answers.

Prompt clarity also improves attention because the model knows which words deserve more focus.
